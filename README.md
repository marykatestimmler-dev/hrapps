# HR Apps Collection

A collection of 20 interactive, research-based HR and workplace tools built with React. Each tool is a standalone HTML file that runs entirely in the browser - no server required.

## Live Demo

Visit: **https://marykatestimmler-dev.github.io/hrapps/**

## Technology

- **React 18** - UI framework (loaded via CDN)
- **Babel** - JSX transformation (loaded via CDN)
- **No build step required** - Each file is self-contained
- **No backend** - All data stays in the browser session

## Privacy

All tools run entirely in your browser. No data is sent to any server.
